    First-Time Setup Sequence¶

    When setting up KrakenHashes for the first time, follow this sequence:

    1. Upload Hashcat Binary (Required First)
    2. Navigate to Admin → Binary Management
    3. Upload a compressed hashcat binary (.7z, .zip, .tar.gz)
    4. Wait for verification to complete

    5. This triggers creation of system preset jobs including the potfile job

    6. Verify Potfile Initialization

    7. Check Resources → Wordlists for "Pot-file" entry
    8. Check Admin → Preset Jobs for "Potfile Run" job

    9. Both should exist after binary upload

    10. Continue with Standard Setup

    11. Upload wordlists
    12. Configure agents
    13. Create hashlists

    Best Practices¶

    Security First

    • Enable MFA for all administrative accounts
    • Regularly rotate API keys and passwords
    • Monitor system logs for suspicious activity
    • Keep all components updated

    Performance

    • Schedule intensive jobs during off-peak hours
    • Monitor agent resource utilization
    • Implement data retention policies
    • Optimize database indexes regularly
